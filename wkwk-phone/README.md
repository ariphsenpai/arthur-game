# wkwkphone — Online Store

🏪 **Toko smartphone online** sederhana & modern.

### Fitur
- 📱 **Katalog produk** — 8 varian HP dengan spesifikasi lengkap
- 🛒 **Keranjang** — tambah/kurang quantity, simpan di localStorage
- 💬 **WhatsApp Checkout** — order langsung via WA
- 🌙 **Dark mode** UI premium
- 📱 **Responsive** — mobile friendly

### Tech Stack
- HTML5 + CSS3 (Inter font, gradient, glassmorphism)
- Vanilla JavaScript (no framework)
- localStorage untuk cart persistence

### Cara Pakai
Buka `wkwk-phone/index.html` di browser langsung.
